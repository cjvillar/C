# 
Source: Explore lessons in C 

compile using:
```C
gcc <program.c>
``
or
```C
gcc -o <out_file_name> <in_file_name.c>
```
For verbose mode:
```C
gcc -o -v <out_file_name> <in_file_name.c>
```
To compile but not link:
```C
gcc -c <in_file_name.c>
```
Create a .o output file that needs to be linked
